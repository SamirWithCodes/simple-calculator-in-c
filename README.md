# Simple Calculator in C

A lightweight and user-friendly command-line calculator built using the C programming language. This project performs basic arithmetic operations and serves as a great demonstration of conditional logic and functions in C.

## Features
- **Addition (+)**: Adds two or more numbers.
- **Subtraction (-)**: Subtracts one number from another.
- **Multiplication (*)**: Multiplies two numbers.
- **Division (/)**: Divides a number with error handling for division by zero.
- **User-Friendly Interface**: Clean and easy-to-navigate console menu.

## Prerequisites
To run this project locally, you need a C compiler installed on your system:
- **Windows**: MinGW / GCC or Code::Blocks
- **macOS**: Xcode Command Line Tools (`gcc`)
- **Linux**: GCC compiler
